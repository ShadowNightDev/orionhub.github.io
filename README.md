## 🚀 Ecosystem

Orion Hub consists of three core applications tailored for different platforms and use cases:

### 📺 Orion+
**The ultimate Android streaming center.**
Experience Live TV, Movies, and Shows through a pristine, ad-free interface built for the modern era.
* **Platform:** Android
* **Category:** Video, Streaming

### 🎵 Sonix Music
**A high-fidelity music experience.**
Organize and play your local library without compromises. Built natively for Windows for maximum performance and integration.
* **Platform:** Windows
* **Category:** Music

### 🎬 AetherStream
**Experimental cross-platform media streaming.**
Pushing the boundaries of what's technically possible in the streaming landscape.
* **Platform:** Windows (Cross-platform)
* **Category:** Video, Streaming

## 🛠️ Built With

* **Vanilla HTML, CSS, JS:** No bloated frameworks, ensuring maximum performance and zero loading times.
* **Google Apps Script:** Used as a proxy to fetch live download statistics and version numbers directly from GitHub Releases without exposing API keys.
* **Modern CSS Features:** Leveraging CSS Grid, Flexbox, CSS Variables, and 3D Transforms (`preserve-3d`) for a highly polished, interactive UI.

## 📈 Live Statistics

The website automatically tracks and displays the following data by communicating with the GitHub API via a secure backend:
* Total Downloads
* Current Version
* Interactive Changelogs (Fetched directly from GitHub Releases)
